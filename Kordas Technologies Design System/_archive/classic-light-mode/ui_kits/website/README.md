# Website UI Kit — Kordas Technologies

A marketing-website recreation. Single-page demo at `index.html` showcases the homepage layout with all key components stitched together.

## Files
- `index.html` — homepage demo (loads all components)
- `Nav.jsx` — sticky top nav with logo, nav links, CTA
- `Hero.jsx` — hero block (eyebrow / display title / blurb / CTA pair / data-bar motif)
- `Capabilities.jsx` — 3-column capability cards (AI · Rapid Prototyping · IoT)
- `CaseStudy.jsx` — featured case study panel (image slot + key results)
- `Process.jsx` — numbered process steps
- `Stats.jsx` — KPI strip on dark background
- `ContactCTA.jsx` — bottom CTA with form
- `Footer.jsx` — multi-column footer with logo + contact + nav

The site uses Polish copy as the brand defaults. Tech terms remain English inline.
